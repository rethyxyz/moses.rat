# moses.rat
`moses.rat` is currently being built.

`moses.rat` is a non-admin user Remote Access Tool targeting Windows 10 machines.

# Modules
* Modules include:
	* Screenshot
	* Command execution
	* Fetch public IP

# Features
* All RAT command data requested is uploaded to a https://mega.nz account you configure
